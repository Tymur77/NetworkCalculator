Working with IP addresses

Introduction

This section describes how to obtain IP addresses from other data types.


Functions

Populates an ip address with values from a string

Produces the first ip address in a network

lastIPAddress

Produces the last ip address in a network

IPAddressFromString

Populates an ip address with values from a string 

int IPAddressFromString(
    IPAddress address,
    const char *string);
Parameters
address

ip address, which will hold the extracted values

string

C-style string, reprsenting the ip address, for example, "192.168.0.1"

Return Value

Returns 0, if the operation was successful, otherwise a non-zero value.

Discussion

The delimiter between the parts of the ip address may be any character, greater than the maximum digit of base 10 (which is 9), or an arbitrary character.

firstIPAddress

Produces the first ip address in a network 

void firstIPAddress(
    IPAddress address,
    Network *network);
Parameters
address

ip address, which will hold the values of the first ip address in the network

network

network, holding the information necessary for the calculation of the first ip address. Ideally created with CreateNetwork.

Discussion

For example, network "192.168.0.0" with mask "255.255.0.0" will have the first ip address equal to "192.168.0.1".

lastIPAddress

Produces the last ip address in a network 

void lastIPAddress(
    IPAddress address,
    Network *network);
Parameters
address

ip address, which will hold the values of the last ip address in the network

network

network, holding the information necessary for the calculation of the last ip address. Ideally created with CreateNetwork.

Discussion

For example, network "192.168.0.0" with mask "255.255.255.0" will have the first last address equal to "192.168.0.254". Note that the directed broadcast address is not the last ip address.
